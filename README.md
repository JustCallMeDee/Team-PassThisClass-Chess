# How to Play
## Step 1: Clone the repo
```sh
git clone https://github.com/JustCallMeDee/Team-PassThisClass-Chess.git
cd Team-PassThisClass-Chess
```
## Step 2: Run the game
```sh
java ChessMain.java
```
